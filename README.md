# Smart Reply Generator

## Overview
A lightweight AI-inspired smart reply system for generating instant message responses.

---

## Features
- Message classification
- Context-aware replies
- Lightweight NLP workflow
- Smart messaging assistance

---

## Files
- `analyzer.py` → classifies message type
- `reply.py` → generates smart replies

---

## Usage

```bash
python reply.py
```

---

## Architecture

```text
Incoming Message
        ↓
Message Analysis
        ↓
Reply Generation
        ↓
Suggested Response
```

## Example Output

```text
Sure, let me check my schedule.
```
